# RouterDash для OpenWrt

Исправленный комплект установщика.

## Что изменено

- GitHub-установщик теперь работает строго по шагам:
  1. выбор языка
  2. на выбранном языке выбор действия: установить или удалить
  3. выполнение выбранной операции
- файлы сначала скачиваются в `/opt/routerdash-installer`
- затем выставляются права доступа
- перед установкой к скачанному `routerdash.py` применяется патч
- добавлен локальный `blinker.py` для OpenWrt-сборок без пакета `python3-blinker`
- исправлен показ IPv4: если фильтр сети не совпал, даш показывает private IPv4 как fallback
- если в конфиге остался старый `192.168.0.0/24`, а LAN у системы уже другая, используется автоматически определённая сеть

## Установка

```sh
wget -O /tmp/routerdash-installer.sh https://raw.githubusercontent.com/t4kyofc/RouterDASH---OpenWRT/refs/heads/main/install-github-template.sh && sh /tmp/routerdash-installer.sh
```

## Рекомендуемый способ после этих правок

Лучше выполнить удаление и затем установку заново через новый лаунчер.
